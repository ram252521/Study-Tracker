# Study Tracker

学習時間を記録・可視化できるシンプルな学習ログアプリです。

## デモ
https://study-tracker-xi-nine.vercel.app/

## 概要

日々の学習内容（科目・時間・メモ）を記録し、
合計時間や科目別の学習量を確認できるアプリです。

## 主な機能

- 学習ログの追加・編集・削除
- 科目別フィルター
- キーワード検索
- 並び替え（新しい順・古い順・時間順）
- 合計・今日の学習時間の表示
- 科目別グラフ表示
- ダークモード切り替え

## 使用技術

- React（Vite）
- Tailwind CSS
- localStorage
- Vercel

## 工夫した点

- フィルター・検索・並び替えを組み合わせた柔軟な表示
- localStorageの遅延初期化で不要な再読み込みを防止
- グラフによる学習量の可視化
- ダークモード対応でUX向上

## 今後の改善

- Laravel連携によるDB保存
- ユーザー認証
- 週・月単位のグラフ
- モバイルUI最適化

## セットアップ

```bash
npm install
npm run dev
```
